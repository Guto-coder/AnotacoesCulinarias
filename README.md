# AnotacoesCulinarias
Um simples bloco de notas para suas melhores receitas
Esse bloco de notas foi desenvolvido a partir de um projeto escolar, com linguagem java e API 21, mantendo compatibilidade com a API 33. De forma totalmente gratuita, com auxilio de tecnologias de inteligência artificial para a programação. O código é iniciante, mas funcional.
